친구가 고된 반복작업으로 고생하고 있길래 파이썬의 셀레니움 기능을 활용해 기자들의 이메일을 스크랩해보는 프로그램을 만들어보았다. 6월14일 기준 5시간 9125개의 이메일을 수집하였으며 카테고리/언론사/이름/이메일 으로 수집해보았다.

엑셀의 파일 크기는 251.64kib, 시간이 있으면 담당 카테고리도 넣어줄려고했으나 시간이 없어서 그대로 넘겨주었음.
