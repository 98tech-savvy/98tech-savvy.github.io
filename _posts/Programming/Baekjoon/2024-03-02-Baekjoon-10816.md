[백준 문제풀이 10816번](https://www.acmicpc.net/problem/10816)
 
## 풀이
딕셔너리로 쉽게 풀이할 수 있다.

## 코드

```py
import sys

input = sys.stdin.readline

N = int(input())
numCard = list(map(int, input().split(" ")))
M = int(input())
myCard = list(map(int, input().split(" ")))

answer = {}

for i in myCard:
    answer[i] = 0

for i in numCard:
    if i in answer:
        answer[i] += 1

for i in myCard:
    print(f"{answer[i]} ", end="")
```
